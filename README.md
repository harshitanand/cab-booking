## Prerequisites ##
1. Git
2. Node.js
3. Yarn

## Installation ##
1. Run yarn to install dependencies.

## Running ##
Run node index.js to start the development server and open up the respective url(Customer,driver,faker or serviceman) like localhost:3000/customer in your browser.
